# MyMusicPlayer
[`MyMusicPlayer`](https://github.com/xiaole0310/MyMusicPlayer)
开源:本项目是一个安卓的在线音乐项目源码，歌曲数据来源于百度音乐盒接口。可以实现歌曲的在线搜索和下载。

# Links
[`App Link`](https://github.com/xiaole0310/MyMusicPlayer)


# Dependencies

* JSON: [`Gson`](https://github.com/google/gson)
